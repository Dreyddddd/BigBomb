# BIG BOOM

Аркадный 2D-шутер на `HTML5 Canvas` с режимами:
- Deathmatch (DM)
- Team Deathmatch (TDM)
- Capture the Flag (CTF)

## Быстрый старт

### Вариант 1: открыть файл напрямую
Откройте `BigBoom.html` в браузере.

### Вариант 2 (рекомендуется): локальный HTTP-сервер
```bash
cd /workspace/BigBomb
python3 -m http.server 8080
```
После этого откройте: `http://localhost:8080/BigBoom.html`.

## Управление
- `WASD` — движение
- `ЛКМ` — стрельба
- `1-7` — переключение оружия
- `TAB` — таблица статистики
- `ESC` — пауза

## Структура проекта
- `BigBoom.html` — UI-слой, экраны и контейнеры HUD.
- `styles.css` — стили интерфейса и экранов.
- `game.js` — игровая логика (рендер, физика, боты, режимы игры, оружие, раунды).

## Советы по разработке
- При изменениях игрового JS проверяйте синтаксис:
```bash
node --check game.js
```
- Для ручной проверки удобнее запускать игру через локальный HTTP-сервер.
